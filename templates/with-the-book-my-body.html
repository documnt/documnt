MY BODY


For "With the book .JPG" I wanted to create an abstract composition using the book without it being visible. At the time I was a student of art history and I had this portable scanner regularly on me. I finally used it more in my artistic practice than in the context of these studies … It creates a file that resembles the A4 in which it automatically inserts the scanned image. This will create ... an interesting enhancement of the scanned object. And this process of automation of creation seduced me. … I like to use this scanner because it allows me a certain freedom in what I scan. … Here a book and my body for example.
